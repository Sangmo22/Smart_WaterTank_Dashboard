# Project Rules

## Git: Auto-push after every code change

After making **any code change** to this project, always:
1. `git add` the modified files
2. `git commit` with a descriptive message following conventional commits format (e.g. `fix:`, `feat:`, `refactor:`)
3. `git push origin master`
